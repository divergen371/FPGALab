# FPGA (Tang Nano 9K) 学習用 電子部品BOM

今後の学習（CPUの完成、周辺回路の実装、IO制御、通信など）において、FPGAのピンに接続して実機動作を確認するために必要・あると便利な電子部品のリストです。

## 1. 基本入出力（必須級）
CPUへの入力と、状態・結果の出力を行うための基本セットです。

| 部品名 | 推奨数量 | 用途・備考 |
| :--- | :---: | :--- |
| **ブレッドボード** | 1〜2枚 | 回路を組むための土台。フルサイズまたはハーフサイズ。 |
| **ジャンパワイヤ** | 1セット | 「オス-オス」と「オス-メス」が両方あると便利です。（FPGAのピンヘッダとブレッドボードを繋ぐため） |
| **タクトスイッチ** | 4〜8個 | リセットボタンや、CPUへの手動データ入力（DIPスイッチの代わり）に使用します。 |
| **LED (発光ダイオード)** | 8〜10個 | 計算結果やCPUのステートメントなどを表示するために使います。赤・緑・黄など色が分かれていると便利。 |
| **抵抗器 (330Ω 〜 1kΩ)** | 10本〜 | LEDの電流制限抵抗として使用します。 |
| **抵抗器 (10kΩ)** | 10本〜 | スイッチのプルアップ / プルダウン抵抗として使用します。 |

## 2. 表示デバイス（ステップアップ）
LED1個1個ではなく、数値や文字を表示するためのデバイスです。

| 部品名 | 推奨数量 | 用途・備考 |
| :--- | :---: | :--- |
| **7セグメントLED** | 1〜2個 | （1桁または4桁）CPUの計算結果を10進数や16進数で表示するために使います。ダイナミック点灯の学習に最適。 |
| **I2C キャラクタLCD** (例: 1602) | 1個 | CPUからの文字出力。I2C通信プロトコルをハードウェアで実装する学習に使います。（※Tang Nano 9KのIO電圧3.3Vに対応しているか確認が必要） |

## 3. 通信モジュール（実践・デバッグ用）
PCとFPGAを通信させ、プログラムのロードやデバッグ出力を可能にする非常に重要なモジュールです。

| 部品名 | 推奨数量 | 用途・備考 |
| :--- | :---: | :--- |
| **USB-TTLシリアル変換モジュール** | 1個 | (FT232, CP2102, CH340など) FPGA側にUART送信/受信モジュールを実装し、PCのターミナルソフト(TeraTermなど)と文字の送受信を行うために使用します。**3.3V対応**のものを選んでください。 |

## 4. その他（あると便利）
| 部品名 | 推奨数量 | 用途・備考 |
| :--- | :---: | :--- |
| **DIPスイッチ** | 1個(4〜8連) | 複数ビットの設定値（例えばALUのオペレーションコードや初期データ）を固定で入力し続けるのに便利です。 |
| **ロジックアナライザ** | 1台 | (安価な24MHz 8chのもので十分) オシロスコープの代わりに、実際の信号線のHigh/Lowの波形をPCで確認するための強力なデバッグツールです。 |

## 購入時のアドバイス
電子工作の初心者向けに販売されている **「Arduino入門キット」や「Raspberry Pi電子工作キット」** のようなパッケージを購入すると、上記の基本部品（ブレッドボード、ワイヤ、LED、スイッチ、抵抗、7セグ、LCD）がすべてセットになっているため、個別に集める手間が省けて非常にコストパフォーマンスが良いです。
